CDI Dashboard

Overview: A CDI Dashboard report helps the CDI Director to view the trend of various productivity and financial metrics over time and analyze the overall performance of the CDI program. The dashboard also helps you group data by week, month, quarter, and year and analyze trends over different time periods and analyze them more easily.

The report provides answers to the following questions.
  • Has the overall review rate improved over time?
  • Has the impact clarification rate improved over time?
  • Have the provider response and agreed rate improved over time?
  • How is my CMI trending month to month?
  • How do I pinpoint the root cause of a downward shift in CMI?
  • Is the CDI program resulting in SOI/ROM category shift?
  • What is the trend around Reimbursement Impact of the CDI program?
  • Am I performing above or below my goals?
  • How do we show the financial impact of the cases post reconciliation?

Navigation: On the Analytics home page left side panel, navigate to CDE One > Executive Summary > CDI Dashboard

CDI Dashboard displays the following:
  • CDI Dashboard Tabular Chart

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the tabular chart.

For filters on all pages refer Filters on all Pages.

For View By option refer View By.

For Footnotes, refer Footnotes.

Default Display: By default, the dashboard displays the data for the inpatient encounters discharged in the last three calendar months. Click the ‘Reset’ button to reset the discharge date and select the required date.

Columns in the chart display the selected View By option.

Total column displays the row-level grand totals for the selected period. For the percentage metrics, this column displays the overall percentage for the selected period.
