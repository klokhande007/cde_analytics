MS DRG Specialty Summary

Overview:

The MS DRG Specialty Summary dashboard provides a high-level comparison of the facility’s service lines by DRG specialty to the hospital’s peer group. DRG specialties include but are not limited to Medicine, Cardiology, Oncology, Pulmonology, and Neurology.

By default, the dashboard displays the data for reconciled inpatient encounters discharged in the last twelve calendar months.

Navigation: On the Analytics home page left side panel, navigate to CDE One > > Peer Group Comparison > MS DRG Specialty Summary.

The MS DRG Speciality Summary dashboard displays the following charts:

  • MS DRG Speciality Summary
  • Secondary DX Summary

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the charts.

For filters on all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

The images are for illustration purposes only.

MS DRG Speciality Summary

This chart displays encounters and Case Mix Index (CMI) by DRG specialty (for example, Cardiology, Oncology, Orthopedics). It helps users see how each specialty contributes to overall performance.

Info Icon (Text on hover): Encounter Status: Reconciled-Impact, Reconciled-No Impact, Auto-Reconciled.

Secondary DX Summary

This chart summarizes encounters based on secondary diagnoses and their severity (with/without CC/MCC). It allows users to understand the impact of secondary diagnoses on DRG outcomes.

Info Icon (Text on hover): Encounter Status: Reconciled-Impact, Reconciled-No Impact, Auto-Reconciled.
