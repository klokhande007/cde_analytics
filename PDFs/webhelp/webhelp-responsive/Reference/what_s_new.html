MS DRG Group Summary Dashboard Update

A new Total column group is added to the MS DRG Group Summary dashboard in the Peer Group Comparison report. It provides an aggregated view of CMI, encounters, and peer group comparison across the selected discharge date range, enabling users to evaluate overall performance.

Also added a footnote, % of Total

 MS DRG Group Summary

Clarifications Detail dashboard Update

The following columns, KPI, and footnote have been added to the Clarifications Detail dashboard:

Columns:
  • Escalated?
  • Escalation Type

KPI: Escalated Clarifications

Footnote: Escalated Clarifications

Note:
When a clarification is escalated (Y), the entire row is highlighted in red font.
