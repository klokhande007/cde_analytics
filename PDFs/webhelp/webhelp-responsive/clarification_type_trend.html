Clarification Type Trend

Overview: The Clarification Type Trend report displays both standard and custom clarifications, including sent, responded, and agreed counts and percentages, all in one table. This table provides a breakdown of Volume, Responded, and Agreed metrics for each month, categorized by clarification type. The values shown for each clarification type include:
  • Vol (% of Total)
  • Responded (%)
  • Agreed (%)
The report provides answers to the following questions.
  • How many clarifications of each type were raised by the CDS every month?
  • How many MCC clarifications were responded by the Physician every month?
  • How many CC clarifications were agreed to by the Physician every month?
  • What percentage of each type of clarifications were raised by the CDS every month?
  • What percentage of each type of clarifications were responded by the Physician every month?
  • What percentage of each type of clarifications were agreed to by the Physician every month?

Navigation: On the Analytics home page left side panel, navigate to CDE One > Executive Summary > Clarification Type Trend.

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the charts.

For filters on all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

Note:

  • Clarification Volume by Type displays the clarification by volume, response, and agreed rates, including the count and percentage of volume, response, and agreed.

  • "Type at Reconciliation" is considered and includes both standard and custom clarification types that are sorted alphabetically in the 'Clarification Volume by Type' tabular chart.
  • Month represents the discharge month.
  • Includes all encounter statuses except 'Review Not Needed'.
  • Includes all clarification statuses except 'Removed' or 'Retracted'.
  • Percentage value is rounded off to 0 decimal place.
Default Display:
  • Report displays the data for 9 calendar months based on the Discharge Date. (Format: Between Date range.)
  • When you reset the discharge date, the encounters that do not have a discharge date assigned are not displayed in the reports. The column with the blank discharge month are not available in the reports.

Clarification Volume by Type

This chart allows the user to analyze the individual count of each clarification type raised by the CDS every month and % calculation of each type with respect to the total clarifications raised.

Table 1. Clarification Volume by Type Details
Columns Description
Vol (% of Total) Displays count and percentage of clarifications generated by the CDS for that clarification type for the selected time period/for every month. Format: : Vol (% of Total) 67 (90%).

Vol: Displays count of clarifications generated by the CDS for the specific clarification type for the selected time period /or against every month.

% of Total: Out of the total clarifications raised by the CDS, the count of clarifications generated for the specific clarification type for the selected time period.

Note:
Clarifications for "Review Not Needed" encounter status are not considered for this calculation.
Responded (%) Displays count and percentage of clarifications responded for that clarification type for the selected time frame. Format: : Responded (%) 2 (22%).

Responded: Displays count of clarifications responded for that clarification type for the selected time frame.

%: Displays percentage of clarifications responded for that clarification type for the selected time frame.

Note:
The following clarifications are not considered in this calculation:
  • Clarifications for encounters with a "Review Not Needed" status.
  • Clarifications marked as "Deleted" and "Retracted" clarifications.
  • Clarifications that do not have a FINAL response.
Agreed (%) Displays count and percentage of clarifications agreed for that clarification type for the selected time frame. Format: Agreed (%) 1 (50%).

Agreed: Displays count of clarifications agreed for that clarification type for the selected time frame.

(%): Displays percentage of clarifications agreed for that clarification type for the selected time frame.

Note:
The following clarifications are not considered in this calculation:
  • Clarifications for encounters with a "Review Not Needed" status.
  • Clarifications marked as "Deleted" and "Retracted".
Table 2. Report Slicers
Slicer Description
Discharge Date
  • Displays the discharge date range.
  • You can select date from the lookup provided.
  • Example: 1/1/2023 - 9/31/2023
  • Default: Last 9 calendar months date range selected by default.
Reset Button
  • With the Reset button, you can reset the discharge date range. Use "Ctrl + Click" to reset the Discharge Date.
Facility
  • Displays both active and inactive facilities in alphabetical order that you have access to.
  • You can select multiple facilities.
  • You can search the facility by Facility ID or Description.
  • Format: Facility Description (Facility ID)
  • Default: ALL
Payer
  • Displays both active and inactive payers in alphabetical order associated with the facilities you have access to.
  • You can select multiple payers.
  • "Unassigned Payer" displays data for the encounters that do not have a payer assigned.
  • You can search the Payer by Payer ID or Description.
  • Format: Payer Description (Payer ID)
  • Default: ALL
Clarification Type
  • Displays both active and inactive type of Clarification in alphabetical order associated with the facilities you have access to.
  • You can select multiple clarification types.
  • You can search the Clarification Type
  • Format: CLIN VAL, CC, MCC, HIV
  • Default: ALL
Unit
  • Displays the list of available units
  • You can select multiple units
  • You can search the unit.
  • Default: ALL
