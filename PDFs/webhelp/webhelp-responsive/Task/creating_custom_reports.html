Creating Custom Reports

To create a custom report, follow the below steps:

  1. From the Predefined section, open a report that you would like to customize.
  2. Make the necessary changes in the report.
    Note:
    • Your account must have editing privileges in order to create/edit custom reports.
    • If modifying a specific visual, you must click somewhere in the visual to select it. Then navigate to the panes to make selections.
    • If you’re not in a specific box or visual on the page, and you just start adding fields from the Fields pane, a new visual is created instead of modifying the existing visual.
  3. Click File and select Save as. This option is not available if you login as a CDS Manager.
    The Report will be saved in the User Defined section on the main page.
    Note:
    • You can also search the report using the search box.
    • By default, the reports are sorted in ascending order (alphabetical order). Click on the reports header to sort the report in descending order.
    • You can modify the custom created report and save it as a new report or overwrite on the same report.
    • You can select the filters/criteria on a report and save it as a custom report to avoid selecting the filters/criteria again.
