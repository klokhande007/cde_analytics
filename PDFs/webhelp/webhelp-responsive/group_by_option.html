Grouping Data for Week, Month, Quarter, Year

For a few results, you can group data by week, month, quarter, or a year. Data can be grouped only when you select the "View By" option as Admit Date or Discharge Date. When you select the "View By" option as Admit Date or Discharge Date, the following filters automatically become available for selection:

  • Group By: You can either select Week, Month, Quarter, or Year.
  • Week Start Day: You can select a week start day. You can select week start day only for Week. For options other than Week, you must manually select Not Applicable. When you select Week, reports are displayed starting from the selected start day, or the start day is considered part of the first week.
    Note:
    These filters are not applicable when you switch to other 'View By' options, such as Facility

The below format is displayed for week, month, quarter, or year:

  • Week: Week start day - Week end day (Example: 7/2/2023-7/8/2023)
  • Month: Month YYYY ( Example: Jan 2024, Feb 2024)
  • Quarter: Jan-Mar 2023, Apr-Jun 2023
  • Year: YYYY (Example: 2022, 2023)
The following are a few effective ways to use the filters for best results:
  • When selecting a week, it is essential to choose a week start day. For instance, if you choose Monday, the reports display data from Monday to the following Sunday. Selecting a start day is mandatory; without it, data will not be displayed.
  • When opting for weekly progress within a month, the data will be displayed weekly starting from the selected day for the entire month.

These filters apply to the following dashboards for the respective categories:

Executive Summary category:
  • Executive Overview
  • CDI Dashboard
  • No Response Impact
Quality Impact category:
  • Quality Impact Overview
  • Quality Impact Summary
Triage and CDS Assistant Usage category:
  • Triage Overview
  • Triage Summary
  • Assistant Usage Overview
  • Ranked Encounters
