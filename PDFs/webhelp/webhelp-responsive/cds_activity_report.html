CDS Activity

Overview: As a CDS, I need a CDS Activity dashboard so that I can view the review activities performed and analyze my productivity.

Navigation: On the Analytics home page left side panel, navigate to CDS General > CDS Monthly Productivity.

The report displays the following:
  • KPI cards: Initial Reviews Per Day, Additional Reviews Per Day, Total Reviews Per Day
  • CDS Activity Tabular Chart
  • This chart displays the data for the activity performed by the CDS in the last 1 week.
  • The user can click the Reset button to reset the dates to last 10 days of activity.
  • The chart displays review activity for every 'Initiating' CDS in the selected date range. Initiating CDS associated with the clarification.
  • Column headers in the chart display the Activity Dates (Last 7 days).
  • Row headers in the chart display the metrics along with the CDS Name.
  • The data in the chart is sorted alphabetically based on CDS.

On the top panel, you can select desired values for the slicers (refer Report slicers) and the corresponding data gets filtered in charts.

For filters on all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

CDS Activity Productivity

Exclude Weekend Activity: When you select this check box, the activities performed on a weekend by the CDS does not appear in the report. This check box is not selected by default.

CDS Activity - KPI Cards

Table 1. KPI Cards
KPI Card Description
Initial Reviews Per Day

Displays the average daily initial reviews performed by the CDS for the selected Activity Date range.

Initial Reviews Per Day = (Total number of Initial Reviews with the Initial Review Date in the selected activity date range/ No. of days CDS worked in the selected activity date range). Rounded to 0 decimal places.

Example: For last 1 week Activity date, this metric displays the average daily initial reviews by the CDS.

Note:

Initial Review Date/Time falls in the selected activity date range.

Encounters with "Review Not Needed" encounter status are excluded from this metric calculations.
Additional Reviews Per Day

Displays the average daily additional reviews performed by the CDS for the selected Activity Date range.

Additional Reviews Per Day = (Total number of Additional Reviews with the additional Review Date in the selected activity date range/ No. of days CDS worked in the selected activity date range). Rounded to 0 decimal places

Example: For last 1 week Activity date, this metric displays out of the total number of days CDS's worked, the count of additional reviews done.

Note:

Additional Review Date/Time falls in the selected activity date range.

Activity Type 'Post-Discharge Reviews' is not considered as an Additional Review by the CDS.

Additional Reviews for Encounters with "Review Not Needed" encounter status are excluded from this metric calculations.
Total Reviews Per Day

Displays the daily total reviews performed by the CDS for the selected Activity Date range

Total Reviews Per Day = (Total number of Initial and Additional Reviews with the Initial Review Date and additional Review Date in the selected activity date range/ No. of days CDS worked in the selected activity date range). Rounded to 0 decimal places

Note:

Additional Review Date/Time falls in the selected activity date range.

Initial Review Date/Time falls in the selected activity date range.

Activity Type 'Post-Discharge Reviews' is not considered as an Additional Review by the CDS.

Initial and Additional Reviews for Encounters with "Review Not Needed" encounter status are excluded from this metric calculations.

Additional Review Types:

  • Clarification creation
  • Clarification modification
  • Working review modification
  • Worksheet creation or modification
  • Possible review creation

  • Reconciliation assessment creation
  • Reconciliation assessment modification

  • Encounter status modification
