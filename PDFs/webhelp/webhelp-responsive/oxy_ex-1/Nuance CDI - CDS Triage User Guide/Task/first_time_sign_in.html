First Time Sign In

This section describes how you can sign-in to Nuance CDE for the first time.

Before you start, your system administrator must set up an account for you.
For first time sign in follow the below steps:
  1. Start your web browser.
  2. Type the URL in the Address field and press Enter or click Search icon.

    The Nuance CDE login screen is displayed.

  3. Type your ID in the User ID field.
    Note:
    If you do not know your User ID, ask your system administrator.
  4. Type the password provided by your system administrator in the Password field.
  5. Press Enter or click Sign In.
    The update password screen is displayed.
  6. Type the password provided by the system administrator in the Old Password field.
  7. Type a new password in the New Password field.
  8. Again, type the new password in the Confirm Password field.
  9. Click Update to change the password or click Cancel to abort.
  10. You are prompted by the CDE Authenticator application to enter a one-time password (OTP). The one-time password (OTP) is sent to your configured contact email address.
    • The one-time password (OTP) is a six-digit number.
    • You may need to wait a few seconds till you receive the email.
  11. Enter the OTP in the text box. The OTP code is valid for only a few minutes.
    • If you select the Remember this device check box, click Submit Code, and use the same browser on the same device to login for the second time, the email verification is skipped (OTP is not sent to the email address). After you enter your user name and password, the system navigates you to the Triage Worklist screen.
    • If you do not select the Remember this device check box, the next time you login, you have to enter the received OTP.
    If you do not receive the code, click Resend Code.
  12. The system navigates you to the Triage Worklist screen.
