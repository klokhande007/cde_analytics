Live Chat Settings

Salesforce supports browsers listed below for Live Agent.

Browser Version
Google Chrome™ Most Recent Stable version
Microsoft Edge Most Recent Stable version

How to enable chat window pop up

While using Live Agent in Microsoft Edge or Google Chrome, you might notice that the chat window does not pop up as a separate window. Rather, it replaces the current window with a chat window. Additionally, a blank small window pops up.

This issue occurs when the domain that you are launching the chat button from and the Live Agent domain, are in different security zones.

For Example:

Domain where chat button is hosted: nuancecdi.com

Live Agent domain: salesforceliveagent.com

Follow these steps for Chrome browser:

  1. Click the icon.
  2. Go to Settings > Privacy and Security > Cookies and other site data > Sites that can always use cookies
  3. Click Add.
  4. Add the Live Agent domains as below:
    • https://nuance.my.salesforce-scrt.com
    • https://nuance.my.site.com
  5. Click Close.
  6. Restart Chrome.

Follow these steps for Edge browser:

  1. Click the icon.
  2. Go to Settings > Cookies and site permissions > Manage and delete cookies and site data > Allow
  3. Click Add.
  4. Add the Live Agent domains as below:
    • https://nuance.my.salesforce-scrt.com
    • https://nuance.my.site.com
  5. Close the dialog box.
  6. Restart Edge.
