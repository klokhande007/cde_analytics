DMO Microsoft Edge Plugin Settings

Follow the steps given below to enable the Dragon Medical One (DMO) Web Extension on Microsoft Edge browser.

Note:
Dragon Medical One (DMO) support on CDE One for Microsoft Edge is only from DMO version 2022.1 onwards.
  1. Open this link in Microsoft Edge browser, https://microsoftedge.microsoft.com/addons/detail/dragon-dmo-dmd-dpa-dl/hhffbmpimhclcocppcaidfhpfeophnde. Click Get.

    A dialog box appears for confirmation.

  2. Click Add extension.
    A confirmation message appears saying the extension has been added to Microsoft Edge.
  3. Click Extension at the top right to view the added extension.
  5. Click Extension > Manage extensions.
  6. On the left, click Extensions, under Dragon (DMO, DMD, DPA, DLA) Web Extension, click Details.
  7. From the Site access section, select option On all sites.
  8. Select the Allow access to file URLs check box.
