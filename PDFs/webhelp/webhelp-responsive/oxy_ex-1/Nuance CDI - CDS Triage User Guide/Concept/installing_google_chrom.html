Installing Google Chrome

In order to start using Chrome as a browser, follow these instructions to install Chrome.

  1. To download Chrome, go to http://www.google.com/chrome.
  2. Click Download Chrome.
  3. Click Accept and Install.
  4. Click Save File.
  5. Double-click the set up lick to begin the downloading process. Click Run.
  6. Once the download is complete, you will be prompted to login and customize the browser. You can choose to skip the process.
  7. On the top panel, click Import Bookmarks Now.
  8. Select the browser from which you want the bookmarks to be imported. Click Import.
