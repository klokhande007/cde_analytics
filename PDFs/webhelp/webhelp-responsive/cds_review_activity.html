CDS Review Activity

Overview: As a CDI Manager, I need a CDS Review Activity dashboard so that I can view the review activities performed by the CDS and analyze their productivity.

You can view the individual and detailed performance of the CDSs to analyze their activity. Select a CDS, right-click, and select Drill through > CDS Performance, the system navigates to the CDS Performance tab. This tab displays all the data for the selected CDS.
Note:
When the CDS Performance tab appears, from the title of the tab, click CTRL + to go back to the previous tab (from the tab you have used the Drill through) command). In this case, it is CDS Review Activity.

Navigation: On the Analytics home page left side panel, navigate to CDE One > Staff Management > CDS Review Activity.

CDS Review Activity displays the following:
  • KPI Cards - Initial Reviews Per Day, Additional Reviews Per Day, Total Reviews Per Day
  • CDS Review Activity chart

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the tabular chart.

For filters all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes .

Default Display: Report will display the data for last 1 calendar month based on the Activity Date. By default, the dashboard displays the data for the inpatient encounters for which activity has been performed in the last one calendar month. Click the ‘Reset’ button to reset the activity date and select the required date. Total column displays the row-level grand totals for the selected period. For the percentage metrics, this column displays the overall percentage for the selected period.

CDS Review Activity
CDS Review Activity - KPI Cards
Table 1. KPI Cards
KPI Card Description Formula/Calculation Exclusion Criteria (if any)
Initial Reviews Per Day

This KPI metric displays the average daily initial reviews performed by the CDS for the selected Activity Date range.

Rounded to 0 decimal places

Example: For last 1 week Activity date, this metric displays the average daily initial reviews by the CDS.

Note:
Initial Review Date/Time falls in the selected activity date range.

Initial Reviews Per Day = (Total number of Initial Reviews with the Initial Review Date in the selected activity date range/ No. of days CDS worked in the selected activity date range )

Encounters with "Review Not Needed" encounter status are excluded from this metric calculations.
Additional Reviews Per Day

This KPI metric displays the average daily additional reviews performed by the CDS for the selected Activity Date range

Example: For last 1 week Activity date, this metric displays out of the total number of days CDS's worked, the count of additional reviews done.

Rounded to 0 decimal places

Note:
Additional Review Date/Time falls in the selected activity date range.

Activity Type 'Post-Discharge Reviews' is not considered as an Additional Review by the CDS.

Additional Reviews Per Day = (Total number of Additional Reviews with the additional Review Date in the selected activity date range/ No. of days CDS worked in the selected activity date range)

Additional Reviews for Encounters with "Review Not Needed" encounter status are excluded from this metric calculations.
Total Reviews Per Day

This KPI metric displays the daily total reviews performed by the CDS for the selected Activity Date range

Rounded to 0 decimal places

Note:
Additional Review Date/Time falls in the selected activity date range.

Initial Review Date/Time falls in the selected activity date range.

Activity Type 'Post-Discharge Reviews' is not considered as an Additional Review by the CDS.

Total Reviews Per Day = (Total number of Initial and Additional Reviews with the Initial Review Date and additional Review Date in the selected activity date range/ No. of days CDS worked in the selected activity date range)

Initial and Additional Reviews for Encounters with "Review Not Needed" encounter status are excluded from this metric calculations.

CDS Review Activity

  • By default, the chart displays the data for the activity performed by the CDS in the last 1 calendar month.
  • The chart displays review activity for every 'Initiating' CDS in the selected date range.

  • Column headers in the chart display CDS in the format Last Name, First Name (CDS ID).

  • The data in the chart is sorted based on CDS alphabetically

CDS/Coder Match

This metric displays the count of Matched Encounters in the selected activity date range.

The total count of encounters with Match Value = Y

Note:
We can determine if there is a encounter for which an activity was performed in the selected activity date range, Same for rate as well.

CDS/Coder Match Logic:

This logic applies to all the reconciled cases. (Reconciled-Impact, Reconciled No Impact, Auto-Reconciled)

This logic applies to all grouper types (including MS-DRG and APR Groupers)

Reconciliation Assessment DRG is not used to calculate the CDS/Coder Match.

CDS/Coder Match logic compares Final Coded DRG to the Possible DRG (Or Working DRG when the Possible DRG is blank).

For the reconciled encounters with Possible DRG, CDS/Coder match is Y when the Final Coded DRG matches Possible DRG

For the reconciled encounters without Possible DRG (Possible DRG is blank), CDS/Coder match is Y when the Final Coded DRG matches working DRG.

Matched Encounters = Count of Reconciled Encounters where Final DRG matches Possible DRG (Working DRG if Possible DRG is Blank)
CDS/Coder Match Rate

This metric displays the CDS/Coder Match Rate for the selected activity date range.

CDS/Coder Match Rate = Count of Reconciled Encounters in which Final DRG matches Possible DRG (Or Working DRG if Possible DRG is Blank) / Count of Reconciled Encounters in the selected date range) *100
