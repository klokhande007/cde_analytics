CMI Breakdown

Overview: CMI Breakdown dashboard allows CDI Directors and Managers to analyze the trend of Medical CMI and Surgical CMI over time.

The report provides answers to the following questions.
  • What is the breakdown of medical vs. surgical cases?
  • What % of reconciled cases were medical?
  • What is the breakdown of medical vs. surgical CMI?
  • How is my Medical CMI trending month-to-month?
  • How is my Surgical CMI trending month-to-month?

Navigation: On the Analytics home page left side panel, navigate to CDE One > Executive Summary > CMI Breakdown.

CMI Breakdown dashboard displays the following:

  • KPI Cards: Overall CMI Before CDI, Overall CMI After CDI, CMI Shift, CMI Shift (%)
  • Medical and Surgical Cases Chart
  • CMI Shift Chart
  • CMI Trend Tabular Chart

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the charts and table.

For filters on all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

Table 1. KPI Cards
KPI card Description Formula/Calculation
Overall CMI Before CDI

This metric displays Overall CMI Before CDI report for a period of 3 months by default.

CMI Before CDI = The sum of Final Coded DRG WTs of all Auto-Reconciled, Reconciled- No Impact, and Reconciled Impact (Impact Type: SOI/ROM Impact) encounters and Reconciliation assessment DRG WT (or Working DRG WT when Reconciliation Assessment Info is not available) of all Reconciled Impact (Impact Type: Both, Financial, (Unassigned (DD< 11/14/2019)) encounters that meet the report criteria / Total count of reconciled cases (Auto-Reconciled, Reconciled-Impact, Reconciled-No Impact) for the selected report criteria.
Overall CMI After CDI

This metric displays Overall CMI After CDI report for a period of 3 months by default.

CMI After CDI= The sum of Final Coded DRG WTs of all Auto-Reconciled, Reconciled- No Impact, and Reconciled Impact (Impact Type: SOI/ROM Impact, Both, Financial, (Unassigned (DD< 11/14/2019)) encounters that meet the report criteria / Total count of reconciled cases (Auto-Reconciled, Reconciled-Impact, Reconciled-No Impact) for the selected report criteria.
CMI Shift

This metric displays the CMI Shift report for a period of 3 months by default.

CMI Shift = The sum of CMI Shift (CMI After CDI - CMI Before CDI) for the Reconciled-Impact (Impact Type: Financial, Both, Unassigned Impact for Discharge Date < 11/14/2019) encounters that meet the report criteria. (Rounded 4 decimal places E.g., 2.1868)/ Total count of reconciled cases (Auto-Reconciled, Reconciled-Impact, Reconciled-No Impact) for the selected report criteria.
CMI Shift (%)

This metric displays CMI Shift (%) for a period of 3 months by default. Rounded to 2 decimal places. E.g. 2.74%.

CMI Shift % = (The sum of CMI Shift (CMI After CDI - CMI Before CDI) for the Reconciled-Impact (Impact Type: Financial, Both, Unassigned Impact for Discharge Date < 11/14/2019) encounters that meet the report criteria) / (CMI Before CDI for Auto Reconciled, Reconciled-No Impact, , Reconciled-Impact (SOI/ROM Impact) Reconciled-Impact (Impact Type: Financial, Both, Unassigned Impact for Discharge Date < 11/14/2019) encounters that meet the report criteria). X 100.

CMI Breakdown - Medical and Surgical Cases Chart

The ‘Medical and Surgical Cases’ chart displays the volume and percentage of Medical and Surgical cases for each month.

  • X-axis: Displays the discharge month in ascending order. (Format: Jun 2020, Jul 2020, etc.)
  • Y-axis: Displays the percentage of encounters.

  • By default, the chart displays the data for 3 calendar months.

  • Info Icon(text on hover): Medical Cases: Encounters with Final Coded DRG as Medical. Surgical Cases: Encounters with Final Coded DRG as Surgical. Encounter Status: Reconciled-Impact, Reconciled-No Impact, Auto-Reconciled.
Note:
Medical encounters are those for which Final Coded DRG is Medical (M) ( if M/S indicator is not available for Final DRG and Final DRG Code = Assessment DRG Code, then Assessment DRG is Medical. if M/S indicator is not available for Final DRG and Assessment DRG, and Final DRG Code = Working DRG Code, then Working DRG is Medical)
Note:
Surgical encounters are those for which Final Coded DRG is Surgical (S) ( if M/S indicator is not available for Final DRG and Final DRG Code = Assessment DRG Code, then Assessment DRG is Medical. ( if M/S indicator is not available for Final DRG and Final DRG Code = Assessment DRG Code, then Assessment DRG is Surgical. OR if M/S indicator is not available for Final DRG and Assessment DRG, and Final DRG Code = Working DRG Code, then Working DRG is Surgical)

  • Displays the discharge date range.
  • You can select date from the lookup provided.
  • Example: 7/1/2020 - 7/31/2020
  • Default: Last 3 calendar months date range selected by default.
Reset
  • With the Reset button, you can reset the admit date range.
  • Tooltip: Use "Ctrl + Click" to reset the Admit Date.
Facility
  • Displays both active and inactive facilities in alphabetical order that you have access to.
  • You can select multiple facilities.
  • You can search the facility by Facility ID or Description.
  • Format: Facility Description (Facility ID)
  • Default: ALL
Payer
  • Displays both active and inactive payers in alphabetical order associated with the facilities you have access to.
  • You can select multiple payers.
  • "Unassigned Payer" displays data for the encounters that do not have a payer assigned.
  • You can search the Payer by Payer ID or Description.
  • Format: Payer Description (Payer ID)
  • Default: ALL
