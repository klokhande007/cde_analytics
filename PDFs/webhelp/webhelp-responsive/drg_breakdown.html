DRG Breakdown

Overview: A DRG Breakdown report helps a CDI Director to analyze top volume DRGs (MS-DRGs and APR DRGs) and target areas for focused attention and financial improvement.

The report provides answers to the following questions:
  • What are the top MS-DRGs and top APR DRGs by encounter volume for a particular month?
  • How many top DRGs (MS-DRG /APR) are medical?
  • How many top DRGs (MS-DRG/APR) are surgical?
  • What is the MDC category associated with the top DRGs?
  • Which DRG categories should be targeted to help improve the financial health?

Navigation: On the Analytics home page left side panel, navigate to CDE One > Executive Summary > DRG Breakdown.

DRG Breakdown displays the following:

  • KPI Card: Total Reconciled
  • DRGs by Volume Tabular Chart

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the charts and table.

For filters all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

Default Display: By default, the DRG breakdown report displays data for Reconciled Inpatient encounters for the CMS (Grouper Type) for last one calendar month based on Discharge Date.
