Live Chat Settings for Chrome Browser

Salesforce supports browsers listed below for Live Agent.

Browser Version
Google Chrome™ Most Recent Stable version
Mozilla® Firefox® Most Recent Stable version
Apple® Safari® 6.x on Mac OS X
Windows® Internet Explorer® 11

Salesforce supports IE-11 browser, but it needs some additional settings which are mentioned below.

How to enable chat window pop up

While using Live Agent in Internet Explorer or Google Chrome, you might notice that the chat window does not pop up as a separate window. Rather, it replaces the current window with a chat window. Additionally, a blank small window pops up.

This issue occurs when the domain that you are launching the chat button from and the Live Agent domain, are in different security zones.

For Example:

Domain where chat button is hosted: nuancecdi.com

Live Agent domain: salesforceliveagent.com

Follow these steps for Chrome browser:

  1. Click the icon.
  2. Go to Settings > Privacy and Security > Cookies and other site data > Sites that can always use cookies
  3. Click Add.
  4. Add the Live Agent domains as below:
    • *.force.com
    • *.salesforceliveagent.com
    • nuance.salesforceliveagent.com
  5. Click Close.
  6. Restart Chrome.
