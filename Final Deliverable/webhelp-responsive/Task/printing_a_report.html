Printing a Report

  1. From the Predefined or the User define section, open a report that you would like to print.
  2. Click the Print Report icon from the top-right corner of the screen.
    The Print dialog-box is displayed.
  3. Set the print settings for your desired print options.
    You can select the quantity of copies desired, page selection locations, page orientation (portrait to landscape and vice versa), and printer.
  4. Click Apply, if you've made changes in the print setting.
  5. Click Print.
