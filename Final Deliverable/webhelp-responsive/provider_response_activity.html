Provider Response Activity

Overview: Provider Response Activity Report enables the CDI Manager to analyze provider responses at the clarification instance level and identify educational opportunities for the physicians, thereby ensuring high provider participation in the CDI program.

The report provides answers to the following questions.
  • Which diagnosis codes are most often in clarifications?
  • Is the content of a clarification resulting in “Not Agreed” clarifications? Or is this an opportunity for educating providers?
  • Do CDSs require significant follow-up to bring closure to a query?
  • Which providers are not responding to the clarifications sent by the CDS team? Which providers are the bottom and top responders to clarifications?
  • What is the average provider response turnaround time? Which providers should I target to help improve response turnaround time?
  • Which providers could benefit from additional education?

Navigation: On the Analytics home page left side panel, navigate to CDE One > CDI Operations > Provider Response Activity.

Provider Response Activity displays the following:
  • Instances by Provider Response Chart
  • Clarifications by Subject Chart
  • Provider Response Summary

On the top panel, you can select desired values for the slicers (refer Report Slicers) and the corresponding data gets filtered in the charts and table.

For filters all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

Default Display: By default, the dashboard displays the data for the inpatient encounters admitted in the last one calendar month. Click the ‘Reset’ button to reset both admit date and discharge date and select the required date.

Provider Response Activity

Provider Response Activity - Instances by Provider Response Chart

  • X-axis: Displays the provider names

  • This chart displays the count of clarification instances for each provider broken down by instance responses. Example: The 'No Response' displays the count of clarification instances for which the response is No Response.

  • Note:

    • The provider is the 'Responded By' provider associated with the instance.

    • Instances with 'Retracted' response are not displayed in the chart.
  • Sort: Sort by Provider with the highest count of instances to the provider with the lowest count of instances (descending order).
  • Exclusion Criteria:
    • Instances for "Review Not Needed" encounter status are not considered for this metric calculation.
    • Instances for Clarifications with Status 'DELETED' or 'RETRACTED' are not considered for this metric calculation.
  • Tooltip: Provider, Provider Response, Instances Responded
  • Info Icon(Text on hover): Encounter Status: Includes all encounter statuses except 'Review Not Needed'. Instance Responses: Includes all instance responses.

Provider Response Activity - Clarifications by Subject Chart

  • The ‘Clarifications by Subject’ chart displays the count of clarifications sent with the status other than DELETED or RETRACTED for each clarification subject.
  • Note: Clarifications may or may not have a FINAL response.
  • Exclusion Criteria:
    • Clarifications for "Review Not Needed" encounter status are not be considered for this metric calculation.
    • Unsent clarifications (Clarifications that do not have a sent date) are excluded from the counts.
  • Tooltip: Tooltip displays Clarification Subject (Format: R52 Pain, Unspecified) , Clarifications Sent (Format: 24 (11%); the percentage value is the '% of total sent' for each subject)
  • Info Icon(Text on hover): Encounter Status: Includes all encounter statuses except 'Review Not Needed'. Clarification Status: Includes all clarification statuses except 'Deleted' or 'Retracted'.

Provider Response Activity- Provider Response Summary

The provider column displays the provider that is Sent To provider or Responded By provider for at least one instance in the selected date range and filter criteria.

Info Icon(Text on hover): Encounter Status: Includes all encounter statuses except 'Review Not Needed'. Clarification Status: Includes all clarification statuses except 'Deleted' or 'Retracted'.

The following columns are displayed for each provider.

The Average Response TAT in hours for each 'Responded By' provider.

Clarification Instances for "Review Not Needed" encounter status are not considered for this metric calculation.

Instances for Clarifications with Deleted and Retracted status are not considered for this metric calculation.

Instance with Response as Pending, Retracted or No response are not considered in this metric calculation.
Table 2. Report Slicers
Slicer Definition

  • Example: 7/1/2020 - 7/31/2020

  • Displays the discharge date range here. You can select required dates from the date lookup.

Reset
  • With the Reset button, you can reset the admit date range.

  • Displays a list of active and inactive facilities that you have access to. The list is sorted alphabetically.

  • Default: ALL
CDS
  • Displays a list of active and inactive CDSs associated with the facilities that you have access to . The list is alphabetically sorted.

  • "Unassigned CDS" displays data for the encounters that do not have a CDS assigned.

  • Default: ALL
