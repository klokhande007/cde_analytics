About Nuance CDE Analytics

Analytics helps you to analyze data and share insights. It provides a 360-degree view of most important metrics in one place updated in real time. Analytics has 4 building blocks: Visualization, Dashboards, Report, and Dataset.

A typical workflow involves all of these building blocks: the application collects data from datasets, creates reports full of visualizations that highlight interesting facts and insights, pins visualizations from reports to a dashboard and then shares the reports and dashboards.

Visualization

A visualization can be referred to as a visual representation of data. This representation can be in the form of a chart, graph, map, or any other interesting thing that you create to represent your data. Visualizations make it easier to interpret the insight.

Types of the visualizations that are available: waterfall, ribbon, treemap, pie, funnel, card, scatter, and gauge etc. Same visualization can be used on many different dashboards.

Dataset

A dataset is a collection of data use to build reports and dashboards. Each dataset represents a single source of data, for example, an Excel workbook on OneDrive. The data in a dataset does not have to come from a single source. It can be a filtered collection of data which you combined from multiple different sources to produce a unique collection.

Same dataset can be used in many different reports. Visualizations created from a dataset can be displayed on many different dashboards.

Report

A report is a collection of visualizations that appear together on one or multiple pages. Reports help you to arrange your visualizations in a way that it tells the story of your data, just the way you want it. All of the visualizations in a report come from a single dataset. A report can be associated with multiple dashboards.

There are two types of report available in the application:
  • Predefined report: The standard reports shipped with application.
  • User-defined report: You can edit a predefined report based on your requirement and save it as a custom report.

Dashboard

A dashboard is a single screen with interactive visuals, text, and graphics. A dashboard content comes from one or more reports and one or more datasets. A dashboard represents a customized view of some subset of the underlying dataset(s). A dashboard is a single canvas that contains tiles, graphics, and text.
