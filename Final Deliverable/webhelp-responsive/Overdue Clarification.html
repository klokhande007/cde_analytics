Overdue Clarification

Overview: The Overdue Clarification Report displays the clarification status overdue and response status pending of a CDS.

The report provides answers to the following business questions:
  • How is the overall performance of the CDS team in terms of review activity?
  • How is the performance of a specific CDS in terms of review activity?
  • How is the overall performance of the CDS team in terms of clarification activity?
  • How is the performance of a specific CDS in terms of clarification activity?
  • Who are my top three CDS performers in terms of the review rate

Navigation: On the Analytics home page left side panel, navigate to CDS General > CDS Productivity > Overdue Clarification.

The Overdue Clarification Report displays the following:
  • KPI card: Encounters with Overdue Clarification
  • Overdue Clarification by Provider Chart
  • Overdue Clarification by Encounter Status Chart
  • Overdue Clarification Tabular Chart

On the top panel, you can select desired values for the slicers (refer Report slicers) and the corresponding data gets filtered in charts.

For Global filters, refer Global Filters.

Default Display: By default, this dashboard displays data for inpatient encounters for which activity was performed by the CDS in the last 1 week.

Overdue Clarification - Overdue Clarification by Provider Chart

  • The Overdue Clarifications by Provider chart helps you to analyze total overdue clarifications for each provider.
  • The Overdue Clarifications by Provider bar chart displays total overdue clarifications grouped by provider.
  • X-axis: Displays the provider name and ID.
  • Y-axis: Displays the count of overdue clarifications.
  • Sort: Data in the chart is sorted in descending order from provider with the highest number of overdue clarifications to provider with the lowest number of overdue clarifications.

Overdue Clarification - Overdue Clarification by Encounter Status Chart

  • The Overdue Clarifications by Encounter Status chart helps you to analyze total overdue clarifications for each status.
  • The Overdue Clarifications by Encounter Status bar chart displays total overdue clarifications grouped by status.
  • X-axis: Displays the Encounter Status.
  • Y-axis: Displays the count of overdue clarifications.
  • Sort: Data in the chart is sorted in descending order from status with the highest number of overdue clarifications to status with the lowest number of overdue clarifications.
  • Data in the visual must be sorted by the Encounter Status with the highest number of overdue clarifications in the descending order.
