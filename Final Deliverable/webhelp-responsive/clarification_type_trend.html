Clarification Type Trend

Overview: Clarification Type Trend dashboard enables CDI Managers to analyze the distribution of the clarification types, response and agreement rates to the clarifications over a period of time.

The report provides answers to the following questions.
  • How many clarifications of each type were raised by the CDS every month?
  • How many MCC clarifications were responded by the Physician every month?
  • How many CC clarifications were agreed to by the Physician every month?
  • What percentage of each type of clarifications were raised by the CDS every month?
  • What percentage of each type of clarifications were responded by the Physician every month?
  • What percentage of each type of clarifications were agreed to by the Physician every month?

Navigation: On the Analytics home page left side panel, navigate to CDE One > Executive Summary > Clarification Type Trend.

Clarification Type Trend displays monthly trend of:
  • Clarification Volume by Type Tabular Chart
  • Clarification Response by Type Tabular Chart
  • Clarification Agreement by Type Tabular Chart

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the charts.

For filters on all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

Default Display:
  • Report displays the data for 9 calendar months based on the Discharge Date. (Format: Between Date range.)
  • When you reset the discharge date, the encounters that do not have a discharge date assigned are not displayed in the reports. The column with the blank discharge month are not available in the reports.

Clarification Type Trend

Clarification Type Trend - Clarification Volume by Type Tabular Chart

This chart allows the user to analyze the individual count of each Clarification type raised by the CDS every month and % calculation of each type with respect to the total clarifications raised.

  • Volume and % of Total columns are displayed for each month separately.
  • Month represents the Discharge Month.
  • Info Icon(text on hover): Encounter Status: Includes all encounter statuses except 'Review Not Needed'. Clarification Status: Includes all clarification statuses except 'Deleted' or 'Retracted'.
Table 1. Clarification Volume by Type Details
Columns Description Formula/Calculation Exclusion criteria (if any) Tooltip
Clarification Type
  • The available 'Type at Reconciliation'.

    Example: CC, Clin Val, MCC, PDX etc.

  • Both Standard and Custom Clarification Types are displayed.
  • Sort Order: Alpha
Volume
  • For each Type at Reconciliation, this column displays the individual count of the clarifications raised by the CDS.
 Clarifications for "Review Not Needed" encounter status are not be considered for this metric calculation.
% of Total
  • For each Type at Reconciliation, this column displays the % of individual clarification type raised by the CDS w.r.t the Total Clarifications raised for each month.
  • % value is rounded off to 0 decimal place.
 Calculation = (Count of individual Clarification Type raised by CDS for one month / Total Count of Clarifications raised by the CDS for that month ) *100
Totals The total count of clarifications sent by the CDS for that month and overall % of Total (100% always).

Clarification Type Trend - Clarification Response by Type Tabular Chart

This chart allows the user to analyze the count and percentage of clarifications responded by the Physician for each clarification type.

  • Response (Count) and Response (%) columns are displayed for each month separately.
  • Month represents the Discharge Month.
  • Info Icon(text on hover): Encounter Status: Includes all encounter statuses except 'Review Not Needed'. Clarification Status: Includes all clarification statuses except 'Deleted' or 'Retracted'.
Table 2. Clarification Response by Type details
Column name Description Formula/Calculation Exclusion Criteria (if any) Tooltip
Clarification Type
  • All the available 'Type at Reconciliation'. (Ex: CC, Clin Val, MCC, PDX etc.)
  • Both Standard and Custom Clarification Types are displayed.
  • Sort Order: Alpha
Response(Count) For each Type at Reconciliation, this column displays the count of sent clarifications with Final response other than Pending, Retracted, or No Response.

Response (count) = Count of clarifications with a Final response as Agreed, Not Agreed, Other Explanation, Alternative Specificity Diagnosis, Discuss, Sent to another Provider or Unable to Determine.

  • Clarifications for "Review Not Needed" encounter status are not considered for this metric calculation.

    "Deleted" and "Retracted" Clarifications are not considered for this metric calculation.

    Clarifications that do not have a FINAL response are not considered in this calculation
Response (%) Percentage of sent clarifications responded by the providers for each 'Type at Reconciliation

Response (%) =(Count of clarifications with a Final response as Agreed, Not Agreed, Other Explanation, Alternative Specificity Diagnosis, Discuss, Sent to another Provider or Unable to Determine/Count of Clarifications Sent) x100. Example: 90%

 Clarifications for "Review Not Needed" encounter status should not be considered for this metric calculation.

"Deleted" and "Retracted" Clarifications are not considered for this metric calculation.
Totals The total responses and overall Response (%) for each month.

Clarification Type Trend - Clarification Agreement by Type Tabular Chart

This tabular allows the user to analyze the count and percentage of clarifications agreed by the Physician for each clarification type.

  • Agreed (Count) and Agreed (%) columns are displayed for each month separately.
  • Month represents the Discharge Month
  • Info Icon(text on hover): Encounter Status: Includes all encounter statuses except 'Review Not Needed'. Clarification Status: Includes all clarification statuses except 'Deleted' or 'Retracted'.
