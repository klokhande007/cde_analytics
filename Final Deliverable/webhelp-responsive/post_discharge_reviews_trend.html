Post-Discharge Reviews Trend

Overview: As a CDI Manager, a Post-Discharge Reviews Trend report helps you to analyze the overall impact of the Post Discharge review tool.

The report provides answers to the following questions.
  • What is the trend around post-discharge reviews?
  • Has the number of opportunities addressed improved over time?
  • Has the number of encounters with Unresolved Clarification opportunity gone down over time?

Navigation: On the Analytics home page left side panel, navigate to CDE Triage > Post-Discharge Reviews > Post-Discharge Reviews Trend.

Post-Discharge Reviews Trend report displays the following:
  • Post-Discharge Reviews Summary Chart
  • Overall Reimbursement Impact Chart
  • Impact Opportunities Identified and Addressed Tabular Chart

On the top panel, you can select desired values for the slicers (refer Report Slicers ) and the corresponding data gets filtered in the charts.

For filters on all pages, refer Filters on all Pages.

For Footnotes, refer Footnotes.

Post-Discharge Reviews Trend

Post-Discharge Reviews Summary Chart

  • The ‘Post-Discharge Reviews Summary ’ chart displays the monthly trend of Encounters with Impact Opportunity, Encounters Reviewed, and Encounters Impacted.

  • X-axis: Displays the discharge month.
  • Y-axis: Displays the count of encounters.
  • Info icon(text on hover): Displays the trend of Encounters with Impact Opportunity, Encounters Reviewed, and Encounters Impacted.
Table 1. Post-Discharge Reviews Summary metrics
Metric Definition
Encounters with Impact Opportunity

The count of encounters with at-least one post-discharge review opportunity identified.

This metric displays the count of encounters presented to the Post-Discharge Review worklist.
Encounters Reviewed

The count of encounters reviewed by the post-discharge case reviewer.

Out of total 'Encounters with Opportunity', the count of encounters with at-least one resolved opportunity.
Encounters Impacted

The count of reviewed encounters for which the post-discharge review resulted in a higher reimbursement.

Out of Encounters Reviewed, the count of encounters in which the Final Billed DRG WT was greater than the Final Coded DRG WT.

Overall Reimbursement Impact Chart

  • The ‘Overall Reimbursement Impact ’ chart displays the monthly trend of overall reimbursement impact in dollars.

  • X-axis: Displays the discharge month.
  • Y-axis: Displays the dollars.
  • Info icon(text on hover): Displays the trend of overall reimbursement impact resulting from shift in the WT between Final Billed DRG WT and Final Coded DRG WT.
Table 2. Overall Reimbursement Impact Metrics
Metric Definition
Overall Reimbursement Impact ($) Overall Reimbursement Impact for a discharge month = Sum of ‘Reimbursement Impact’ values of all Post-Discharge Review cases for that month.
  • 'Reimbursement Impact' for an encounter is calculated using the below formula:
  • Reimbursement Impact = Variance x blended rate. (Rounded to 0 decimal places. E.g. $5,234)
  • Variance = Final Billed DRG WT - Final Coded DRG WT

Impact Opportunities Identified and Addressed

  • Impact Opportunities Identified and Addressed chart displays the count of opportunities identified and addressed for each opportunity type in a given month.
  • 'Impact Opportunity' column displays the description of the Opportunity Type. Example: Resequencing PDx
  • Identified and Addressed columns are displayed for each month.
  • Info icon(text on hover): Displays the trend of post review impact opportunities identified and addressed by the reviewer for each impact opportunity type.
Table 3. Report Slicers
Slicer Description
Discharge Date
  • Displays the discharge date range.
  • You can select date from the lookup provided.
  • Example: 7/1/2020 - 7/31/2020
  • Default: Last 3 calendar months date range selected by default.
Reset Button
  • With the Reset button, you can reset the admit date range.
  • Tooltip: Use "Ctrl + Click" to reset the Admit Date.
Facility
  • Displays both active and inactive facilities in alphabetical order that you have access to.
  • You can select multiple facilities.
  • You can search the facility by Facility ID or Description.
  • Format: Facility Description (Facility ID)
  • Default: ALL
Payer
  • Displays both active and inactive payers in alphabetical order associated with the facilities you have access to.
  • You can select multiple payers.
  • "Unassigned Payer" displays data for the encounters that do not have a payer assigned.
  • You can search the Payer by Payer ID or Description.
  • Format: Payer Description (Payer ID)
  • Default: ALL
Opportunity Type
  • Displays the list of Opportunity Types. This list is sorted alphabetically.
  • You can select multiple opportunities.
  • Format: Opportunity Description

    Example: Resequencing PDx, Unresolved Clarification, PSI/HAC Status review, etc.

  • Default: ALL
No Further Action Required
  • Displays a list which includes Y, N, ALL.
  • Y: Displays data for the encounters for which the reviewer selected ‘No Further Action Required’ checkbox on the Post-Discharge Review Checklist screen.
  • N: Displays data for the encounters with ‘No Further Action Required’ checkbox unchecked on the Post-Discharge Review Checklist screen.
  • ALL
  • Default: ALL
Reviewer

Displays the list of active and inactive users with the Post-Discharge Reviewer or Post-Discharge Manager role assigned.

Format: Last Name, First Name (User ID)

Table 5. Footnotes
Formulas Text on hover
Encounters with Impact Opportunity The count of encounters with at-least one post-discharge review opportunity identified.
Encounters Reviewed

Encounters Reviewed: The count of encounters reviewed by the post-discharge case reviewer.

%Reviewed = (Encounters Reviewed by the post-discharge case reviewer/ Encounters with Impact Opportunity) x 100). E.g. 10%
Encounters Impacted

Encounters Impacted: The count reviewed encounters for which the post-discharge review resulted in a higher reimbursement.

%Impacted = (Encounters Impacted / Encounters with Impact Opportunity) x 100) E.g. 10%
Identified Identified: Count of potential post review impact opportunities identified for the encounter. E.g. Resequencing PDx, PSI/HAC Status Review.
Addressed Addressed: Count of post review impact opportunities addressed/reviewed by the post-discharge case reviewer.
Reimbursement Impact

Post-Discharge Review Reimbursement Impact = (Variance x Blended Rate) E.g. $ 15,234

Negative ‘Reimbursement Impact’ is displayed in red font.
