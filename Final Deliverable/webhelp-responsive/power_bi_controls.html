Power BI Controls

Table 1. Power BI Controls in Report
Option Description
Drill Up Displays one level up information in the hierarchy. For example, if you are in month view, you can drill up for quarter view and then year view.
Go to Next Level in the Hierarchy Displays one level down information in the hierarchy. For example, if you are in year view, you can drill down to quarter view and then month view.
Expand all down one level in the hierarchy

Expand adds an additional hierarchy level to the current view. For example, if you are looking at the year level, you can expand and add quarter, and month detail.
Drill Down Click on the Drill Down button and then select the graph to view the information one level down in the hierarchy. For example, if you click on year to drill down, it will show quarters in the year, then further months in the year, then finally days.
Focus Mode Displays the table/bar graph/line graph in zoomed view.
More Options
Export data Exports the table data to .xlsx or .csv format.
Show data Displays all the data in table format.
Spotlight Highlights the table where icon is clicked.
Sort descending Sorts the table data in descending order.
Sort ascending Sorts the table data in ascending order.
Sort by Sorts the data in ascending order according to the selected option/column.
Note: All the table columns are displayed under this option.
